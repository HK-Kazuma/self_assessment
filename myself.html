FILE 01 ／ 幼少期
🎮

マイクラの何にはまっていたの？何が楽しかったの？

一部解明
「マイクラの何にはまっていたの？何が楽しかったの？」

わかっていること

  • 小6、友達がやっているのを見て「良さそう」と思い開始。プレイ形態は一人・友達の家・オンライン通信の3パターン
  • サバイバル、建築、インフラ整備（村人を使った収穫機・丸石製造機）、冒険（洞窟・ネザー要塞探検）まで幅広く遊んでいた
  • 神社・寺・コンビニ・城・図書館・学校・道路・信号機など、現実にある建物の再現が好きだった
  • 建築の工程（整地→資材集め→建築→装飾）のうち、一番時間をかけ夢中になっていたのは装飾。こだわるほど時間を忘れ、「ここも作りたい」と理想がどんどん広がる感覚があった
  • 完成品を友達やネットの友達に見せるワクワクもあったが、誰にも見せなくても同じくらい夢中になれた
  • 小学校〜中学の「秘密基地ごっこ」（竹や木で椅子・パチンコ・発射装置作り）でも同様の感覚があった
  • 材料不足・落下死・敵襲撃といった壁は面倒に思いつつ辞めずに対策し、乗り越えると達成感を得ていた
  • 現在のアプリ開発ではコードのほとんどをAIが書き、「自分の手で徐々に形にするワクワク」より「意思決定者としてどれだけ満足いくものを作れるか」に重心が移った、という自覚がある

今の解釈

一番好きだったのは「何もないところから徐々に形になっていく過程」、特に装飾で理想がどんどん広がっていく感覚。完成させること自体に完結した楽しさがあり、他者からの承認は必須ではなかった。

マイクラの苦労（材料不足・敵襲撃など）は自ら求めて選びに行ったものではなく、作りたいものを実現する過程で自然発生したもの。「苦労の設計者」とマイクラの経験は別物。 ― 2026.07.08 セッションより

ここから、「苦労の設計者」（後述ROOT 02）とは独立した、「何もないところから形にしていく・作り込みたい」という欲求の存在が新たに見えてきた。直接手を動かして作る形（マイクラの装飾、秘密基地作り）から、AIに指示を出し理想を実現する「意思決定者」としての形（アプリ開発、HTML整理での指示出し）へと、表現方法を変えながら持続している。

検証中：新しく見えてきた問い

この「作り込みたい欲求」は、以前見つかった「未踏の地への冒険欲求」（ROOT 03・お金の分析より）と根っこで繋がっているのか、独立した2つの欲求なのか。また「苦労の設計者」という価値観の形成時期はまだマイクラ以前（幼少期）まで遡れておらず、今回見つかった「作りたい欲求」がその形成に関与しているかも未検証。

FILE 02 ／ 高校1年 冬
📖

ジェダイの哲学で、何によって、何がきっかけで将来を考えるようになったの？

解明
「ジェダイの哲学に何によって、何がきっかけで将来を考えるようになったの？」

わかっていること

  • それ以前から、父の影響によるスターウォーズ・ゲームの世界、NCS楽曲を通じた「意味はわからないが英語ってかっこいい」という感覚、中学での洋画・洋ドラマを通じた「個人が尊重される文化」への憧れが、行動に移せないまま蓄積していた
  • 高1冬、『ジェダイの哲学』でルークが「農家を継ぐ人生から抜け出したい」と願う心情に自分を重ねた
  • ヨーダの「できるかできないかじゃない、やるかやらないかだ」という、思い込みを手放す教えに触れた
  • そこで「英語を話せるようになって、色んな国の人と交流するんだ」という決意に変わった

今の解釈

この本は憧れを生んだきっかけではなく、すでに溜まっていた「かっこいいから憧れる」という受動的な感情を、「思い込みを手放して、やる」という行動原理に変換する装置として機能した。

本のマインドマップを作った際、「手放す」という言葉が先入観・主体性・価値観の3箇所に現れる構造を発見した。自分の回避動機（後悔したくない）をダークサイド側の「欲望＝短期的な原動力」と重ね、工業高校時代の二極的な見方から大学受験を経た「人それぞれ」への変化を、本の「先入観を手放す」構造そのものとして言語化した。 ― 2026.07.08 セッションより

さらに掘り下げ、「主体性」には2層あるという整理に至った。行動としての主体性＝苦労する環境を自分で握りにいくこと（TOEICクラス直談判など）。内心としての主体性＝価値観を手放して受け身になること（バイトのピーク時に「あ、今イライラしてるな」と気づくメタ認知のような働き）。前者は行動、後者は考え方であり、階層が違うだけで矛盾しないという仮説に至った。

検証中：残っている部分

本を再読した上で、「文化を経験したい」という核がどの一節にもっと具体的に紐づいているのかは、まだ手つかず。また、「忙しさによる今への集中」が本当の意味での「手放し」なのか、それとも未来の心配を見ないで済む環境を作っているだけの意識逸らしなのかは、本人も「正直わからない」と保留した最重要ポイントとして残っている（→FILE 04の課題と直結）。

FILE 03 ／ 高校1年 〜 現在
💰

「お金を稼ぎたい」「海外で暮らしたい・働きたい」になったのはなぜ？　何で叶えたいの？　叶えた先の何を求めているの？

解明
「『お金稼ぎたい』『海外で暮らしたい・働きたい』になったのはなんで？そもそもそれを何で叶えたいの？叶えた先の何を求めているの？」

わかっていること

  • 高1冬、ジェダイ哲学をきっかけに「自分は何がしたいのか」を自問し、この2つの願いを初めて言語化した
  • 小中学時代、自転車で20〜30km先の川の上流・廃村・お城・寺などへ、思い立ったらすぐ1人でも向かっていた
  • ドラクエの「ルーラ」（一度行った場所にしか使えない）システムが好きで、未踏の地への到達プロセス自体を楽しんでいた
  • 大学1回生の国際ボランティア説明会で「渡航は自腹」と知り断念。海外留学も資金を稼ぎきれず実現しなかった
  • 一方でDiscord語学交流→対面→国内旅行では、お金をかけない工夫で「海外の友達と話したい」という目的を達成できた

今の解釈

お金は目的ではなく手段。目的の達成に他の手段がないと感じる場面でだけ強く立ち上がる、条件付きの欲求だとわかってきた。

幸せはお金では買えない。でも幸せを得る機会は買えると思う。人と会う機会はお金で買えるから。 ― 2026.07.07 セッションより

叶えた先に求めているものの仮説：長年遠くから見つめてきた文化を、実際にこの身で経験すること。初対面でもフレンドリーな距離感、個人が尊重される社会を、自分の生活として味わいたいという願い。

「代替策が効く／効かない」を分ける基準についても一段掘り下げが進んだ。分かれ目は「手段そのものより目的（真に欲しかったもの）を優先したかどうか」。渡航・留学は「海外に行く」という手段自体にこだわり資金不足で断念したが、Discord語学交流→国内旅行は「海外の友達と話したい」という目的を優先したから、お金をかけない代替策にたどり着けた。振り返る中で、奨学金・プログラム・招待など、お金以外のルートもあり得ると気づいた。

検証中：残っている部分

この核自体、本人も「言葉にするのはまだ難しい」と認めている仮説段階。また「目的を優先したかどうか」という基準はまだ言葉として粗く、「お金がないと無理」という思い込み自体をどこまで崩せる余地があるのかは、次回への持ち越し。

FILE 04 ／ 高校1〜3年
🔥

なぜ周りに流されず受験勉強を選択するほど、未来にエネルギーが湧いたの？

解明
「なんで周りの環境には流されず受験勉強を選択するほど、未来にエネルギーが湧いたのはなんで？なんでそんなに未来を叶えたいと思ったの？」

わかっていること：4層の構造

  • ①構造的孤独 ― 工業高校は「大学受験する」が少数派の環境。「周りが動くから自分も動く」が機能せず、一人で動く筋肉がついた
  • ②劣等感の転換 ― 中学時代、学力の高い同級生・部活仲間に囲まれて生まれた劣等感を、受験期に「見方を変えて燃料にする」方向へ転換した経験があった（令和6年1月27日）
  • ③競争の軸の移動 ― 受験期は「見えない他の受験生」との競争、大学入学後は「過去の自分」との競争へと軸が移った
  • ④後悔への恐怖 ― 父親（中卒）の姿を「勉強しなかった未来」の実像として重ね、「絶対こうなりたくない」という回避型動機が非常に強く作用した

今の解釈

意志の強さや気合いではなく、環境・転換体験・競争観・恐怖という4つの積み重ねが、不快感を行動エネルギーに変える回路を作っていた。回避型動機（悪い状態を避けたい）は接近型動機（良いものを得たい）より短期的に強く作用する。

暖かくてぬるい、コンフォートゾーンに浸ったままでいいのか。アグレッシブな人生の方がワクワクする気持ちもある。 ― 令和7年11月24日 ジャーナルより

直近のセッションでは、この回避型エンジンの現在地についてもう一段仮説が進んだ。「今、忙しくしていることが未来の心配を減らしているのかもしれない」という自己観察に対し、これが本当の意味で回避動機を「手放した」結果の集中なのか、単に未来の心配を見ないで済む環境を作ることで管理しているだけの意識逸らしなのかは、本人も「正直なところわからない」と保留している。

検証中：残っている部分

この回避型のエンジンは長期的にどこまで持続するか未検証。「お父さんよりはるかに先に来た」と感じた瞬間、このエンジンが弱まる可能性があり、そこから先の接近型動機（本当に向かいたいもの）への移行が今後の課題として残っている。次の具体的な検証ポイントは、忙しさが途切れた瞬間（長期休み・就活の待ち時間など）に未来の心配がどう振る舞うか。

FILE 05 ／ 幼少期〜中学
🎧

勉強が嫌いな中で、なぜ英語だけは好きだったの？

解明
「勉強が嫌い中で、なんで英語は好きなの？」

わかっていること

  • 小学生時代、父の影響でスターウォーズとゲームを好きになった
  • ゲーム実況・キル集動画でNCS（海外の無料BGM）の曲に触れ、意味は分からないまま「英語ってかっこいい」という感覚が音の響きから生まれた
  • 同時期、公文式で発音重視の英語学習を開始し、日本語英語化しない土台ができた
  • 小6頃、Fortniteで英語圏プレイヤーとチャット・ボイスチャットで交流する小さな実践があった
  • 中学の英語授業冒頭で流れる洋楽をきっかけに洋画・洋ドラマへ関心が拡大し、「初対面でもフレンドリー」「個人が尊重される」文化への憧れが芽生えた

今の解釈

英語は最初から「勉強」として始まっていない。テストのための知識ではなく、娯楽に混ざった「音」として関わり始め、次第に「文化そのもの」への憧れに育っていった。だからこそ、他の教科と同じ「勉強」という枠に入らず、嫌悪感の対象にならなかったと考えられる。

検証中：残っている部分

「個人が尊重される文化への憧れ」が、日本の集団主義的な環境への違和感の裏返しなのか、それとも独立した純粋な憧れなのかは、まだ切り分けられていない。

FILE 06 ／ 高校2〜3年
🏗️

揺らいだということは、建築士になること以上に大事にしていた価値観があったはず。それは何か？

解明
「揺らぐということは、建設で働くこと（建築士になること）以上に自分が大事にしている価値観があったから、未来の変更があったと思うが、その価値観ってなんだろうか？」

わかっていること

  • 高2の建設インターンで、現場監督・職人の働く姿が「やりがい」ではなく「生活のためにやっている」ように見え、冷めた感覚が生まれた
  • 本人が期待していたのは「主体性・誇りを持って働く姿」。そのギャップが最初のヒビになった
  • 現場監督本人からも「現場監督はやめて設計・製図の方がいい」と言われる一方、当の製図（手描き）は苦手だった、という「二重の行き止まり」
  • 同時期、大学ではTOEICクラスへ自ら直談判して編入、英語コミュニティでも段階的に負荷の高い環境に自ら踏み込んでいった

今の解釈

建設現場は、設計者・現場監督・職人の間で板挟みにされ意見を通しにくく、働く環境そのものを選べず与えられた条件に耐えるしかない構造だった。一方でTOEICクラス直談判や英語コミュニティでの環境構築は、すべて「自分から苦労する環境を設計・獲得しに行く」行動だった。

「自分が苦労する環境を、自分で設計・獲得できること」という価値観。建築士という職業そのものへの憧れより、この「苦労の設計者でいられるかどうか」が上位に置かれていた。 ― self-analysis-fukuhonna-kurou.md（2026.07.06–07）

「不本意な苦労」の正体は苦労の量そのものではなく、その苦労が自分の設計によるものかどうか。ITには「劣悪な環境を技術で変える」という、自分が環境の設計に関われる余白が感じられたことが、建築からの転向を後押しした。

検証中：残っている部分

「苦労の設計者でありたい」という価値観そのものが、いつ・何をきっかけに形成されたのかは未検証。FILE 01（マイクラ）の掘り下げにより、「苦労の設計者」とマイクラの経験自体は別物だと判明した一方、そこで新たに見つかった「何もないところから作り込みたい欲求」がこの価値観の形成に関与しているかどうかは、まだ接続できていない。
